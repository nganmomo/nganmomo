USB expansion and input x4

The two modules are combined together to form a simple 4 input/output PLC. It supports all features except analog I/O.



Four USB expansions and user define program chip

The main board consists of two MCUs, ESP32 S3 and RP2040, the second MCU is used for user programs. Users can choose from hundreds of carefully designed or user-predefined functions. The board also comes with 4 USB bus extenders and 1 USB power entry in a 4-inch square box.




Four relays

The system looks complex, however, everything is ready to use and the system can be set up very quickly.




Four relays with MQTT broker

Since the relay board contains an MCU, it is still capable of performing more tasks. Therefore, four relay boards have MQTT broker functionality. The broker is mainly a server LAN, not far from the main board.




Mqtt relays

The Mqtt relay board is the same as other relay boards, except that it is connected via WIFI mqtt.




Solid state Relays

The board is like this, just using solid state relays instead of mechanical relays.

Function in Pick icon screen

(1) Use the mouse left click to select, and the icon cannot be dragged to the editing area. Move the mouse to the editing area and left-click.

(2) Right-click the selected icon, and a help menu will pop up on the right. The help menu must be closed to return to the editing window.

(3) You can select Blank Icons to delete unwanted icons.
