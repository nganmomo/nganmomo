Hardware requirement

Our intention is to support all relay boards or control boards on the market, however, these products must meet minimum safety standards.

These boards shall:

(1) Be suitable for mounting, with suitable mounting points for screws.

(2) The power inlet adopts C high current.

(3) Suitable for installation on ordinary plastic boxes.

(4) Original brand components should be used.


Tools for firmware update

Firmware updates require any USB to serial cable, the start button needs to be pressed to enter boot mode, the PLC also supports some cables that can automatically enter boot mode.

Setup hardware

Power up the PLC, usb cable connect to computer

Run any uart program, like putty, arduino IDE Power up the PLC, usb cable connect to computer

Yes, but must open the firewall in router, or just work at LAN

Can I use phone to control via Mqtt broker.

Yes, you need to install a third-party MQTT client App, some encryption brokers may not be compatible.
